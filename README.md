# rails_tutorial
技術バイトで使うことになったのでruby on rails のチュートリアルを走る

[Ruby on Rails チュートリアル Michael Hartl](https://railstutorial.jp/chapters/beginning?version=5.1#cha-beginning) 著

## Usage
```
$ bundle install
$ rails db:migrate
$ rails test
$ rails server
```
